# AI-CLI

A Terminal AI Agent running on Groq with tool-use capabilities.

## Features
- Execute shell commands.
- Create/Update files.
- Persistent session management.
- Powered by Groq Llama 3 models.

## Installation

### Via Homebrew
```bash
brew install Anay-Rustogi/tap/ai-cli
```

### Via Pip
```bash
pip install .
```

## Usage
Run the command:
```bash
ai-cli
```
Follow the instructions to set up your Groq API key.
